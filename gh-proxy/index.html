GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的，更多用法、clone加速请参考这篇文章。

release、archive使用github.com.cnpmjs.org加速，文件会跳转至JsDelivr

注意，不支持项目文件夹

分支源码：https://github.com/example/project/archive/master.zip

release源码：https://github.com/example/project/archive/v0.1.0.tar.gz

release文件：https://github.com/example/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/example/project/blob/master/filename
